# Proyecto Final Backend

Aplicación web desarrollada con Node.js, Express y MongoDB Atlas para la gestión de productos de maquillaje.

## Tecnologías utilizadas

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- HTML
- CSS
- JavaScript
- Postman


## Funcionalidades

### Usuarios
- Registro de usuarios
- Inicio de sesión
- Encriptación de contraseñas
- Autenticación con JWT

### Productos
- Crear productos
- Obtener productos
- Editar productos
- Eliminar productos

### Frontend
- Interfaz responsive
- Login y registro
- Dashboard de productos


## Instalación

Clonar repositorio:

bash
git clone https://github.com/andreadiaz519/Proyecto-Final-Backend.git

Instalar dependencias:

bash
npm install


## Variables de entorno

Crear archivo `.env`

.env
PORT=3000
MONGO_URI=tu_uri_mongodb
JWT_SECRET=tu_secret


## Ejecutar proyecto

bash
npm run dev


Servidor:

bash
http://localhost:3000
